Controlez vos données personnelles

Le texte de présentation du site

TraceBack est un projet visant à vous redonner le contrôle sur vos données personnelles sur internet. L'extension TraceBack enregistrera localement pour vous l'ensemble des site auxquels vous avez donné des informations, ainsi que les types d'informations que vous avez laissé. Il vous sera donc possible de retracer l'ensemble de vos données dans la jungle du web.
